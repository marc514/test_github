### 功能介绍
暂时(v1.0)仅支持下载单音声、用户上传音声以及用户喜欢的音声，后续版本可能会添加下载歌单、频道音声的功能。

### 演示
输入音声/用户 id ，注意选择是单音声还是用户上传/喜爱音声列表，点击搜索或者快捷键 `Ctrl + Enter`，下载快捷键 `Ctrl + L`

详细说明见 [B 站投稿视频](https://space.bilibili.com/9272615/#/)
